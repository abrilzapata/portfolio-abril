Proyecto

Conversor de divisas

El proyecto consiste en el desarrollo progresivo de un conversor de divisas en Python, comenzando con una versión inicial ejecutada por consola, luego evolucionando hacia una aplicación de escritorio con Tkinter y finalmente hacia una interfaz web desarrollada con Flask. La aplicación permitirá ingresar un importe, seleccionar una moneda de origen y una moneda de destino, para obtener el valor convertido utilizando tasas de cambio en tiempo real mediante una API externa.

📌 Descripción

Este proyecto busca construir una solución práctica para convertir importes entre diferentes monedas, aplicando conceptos de programación, consumo de APIs, validación de datos y diseño de interfaces. La primera versión se desarrollará en CMD/consola, enfocada en la lógica principal de conversión. Luego se incorporará una interfaz gráfica de escritorio con Tkinter, mejorando la experiencia de usuario. Finalmente, se desarrollará una versión web con Flask, permitiendo ejecutar el conversor desde un navegador. El proyecto tiene un enfoque incremental, donde cada etapa mejora la anterior y permite comparar distintos tipos de implementación: consola, aplicación de escritorio y aplicación web.

🎯 Objetivo

Desarrollar una aplicación funcional de conversión de divisas que permita practicar programación en Python, integración con APIs externas y diseño de interfaces de usuario, evolucionando desde una solución básica por consola hasta una aplicación web más completa y visual.

⚙️ Tecnologías utilizadas

  • Python
  • CMD/Consola (primera versión para validar la lógica de conversión)
  • Flask (framework para desarrollar la versión web.)
  • Tkinter (Interfaz gráfica de escritorio)
  • Request (librería para consumir la API de tasas de cambio.)
  • JSON (Formato de respuesta utilizado por la API)

🚀 Funcionalidades principales

  • Ingreso de importe a convertir
  • Selección de moneda de origen
  • Selección de moneda de destino
  • Versión inicial por consola.
  • Versión de escritorio con Tkinter.
  • Versión web con Flask.

🧠 Aprendizajes

  • Desarrollo de interfaces gráficas en Python
  • Consumir una API externa desde Python.
  • Integración de librerías externas
  • Diseño de aplicaciones orientadas al usuario

Funcionamiento

Cómo funciona el sistema

La calculadora combina una interfaz gráfica con lógica matemática para interpretar, resolver y visualizar operaciones de forma dinámica.

1. Ingreso de operaciones

El usuario interactúa con una interfaz visual similar a una calculadora científica real, ingresando operaciones matemáticas mediante botones personalizados.

2. Procesamiento lógico

La aplicación interpreta la expresión ingresada, transforma la sintaxis matemática cuando es necesario y ejecuta el cálculo utilizando lógica programada en Python.

3. Visualización del resultado

El resultado se muestra en una pantalla tipo LCD, manteniendo una experiencia visual coherente con el diseño de una calculadora física.

4. Graficación de funciones

El sistema también permite ingresar funciones matemáticas y visualizarlas gráficamente, integrando cálculo numérico y representación visual mediante Matplotlib.
