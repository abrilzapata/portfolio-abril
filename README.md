# Portfolio Profesional | Abril Zapata

Portfolio web desarrollado para presentar mi perfil profesional, mis proyectos y mi evolución en el aprendizaje de herramientas tecnológicas aplicadas a procesos, datos y experiencia de usuario.

## 🌐 Sitio publicado

[Ver portfolio online](https://abrilzapata-portfolio.netlify.app/)

## 🎯 Objetivo

Diseñar y publicar un sitio web profesional que permita centralizar mi presentación, mis proyectos y los aprendizajes obtenidos durante mi formación técnica y funcional.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- GitHub
- Netlify

## 🚀 Funcionalidades

- Página principal de presentación profesional.
- Sección de proyectos destacados.
- Páginas internas para cada proyecto.
- Galerías de imágenes.
- Navegación entre secciones y proyectos.
- Diseño visual personalizado.
- Publicación online mediante Netlify.

## 📂 Proyectos incluidos

### 🧮 Calculadora científica en Python

Aplicación de escritorio desarrollada en Python con interfaz gráfica y visualización de funciones matemáticas.

### 📊 People Analytics Dashboard

Proyecto visual orientado al análisis de indicadores de Recursos Humanos, con foco en métricas de dotación, movimientos de personal y visualización de información clave.

### 💱 Conversor de Divisas — En desarrollo

Aplicación planificada para convertir importes entre diferentes monedas, evolucionando desde una versión por consola hasta una aplicación de escritorio con Tkinter y una versión web con Flask.
