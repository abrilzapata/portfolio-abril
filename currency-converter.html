PROYECTO WEB

Conversor de Divisas

Aplicación web desarrollada para convertir importes entre diferentes monedas mediante una interfaz simple, clara y orientada al usuario.

HTML5 CSS3 JavaScript UX
CONVERSIÓN

Convertí un importe

Demo educativa
Ingresá un importe y seleccioná las monedas para ver el resultado.

Las tasas utilizadas son demostrativas y no representan valores financieros en tiempo real.

🎯

Objetivo del proyecto

Practicar lógica de programación, manipulación del DOM, validación de formularios y diseño de interfaces web aplicadas a un caso de uso cotidiano.

⚙️

Funcionalidades

  • Ingreso de importe a convertir.
  • Selección de moneda origen y destino.
  • Conversión estimada con tasas internas.
  • Botón para invertir monedas.
  • Validación de datos ingresados.
🧩

Tecnologías utilizadas

  • HTML5 para la estructura.
  • CSS3 para diseño visual responsive.
  • JavaScript para la lógica de conversión.
  • Diseño orientado a experiencia de usuario.

Próximas mejoras

Como evolución futura, el proyecto puede integrarse con una API de cotizaciones para consultar tasas actualizadas y ampliar el conjunto de monedas disponibles.
